# SwiftStock - Inventory Management Dashboard

SwiftStock is a modern web application designed for retail businesses to monitor stock levels, calculate inventory valuation in real-time, and manage restocks efficiently.

## 🚀 Key Features
- **Real-time Dashboard:** Visual cards for Total Valuation, Low Stock Alerts, and Category counts.
- **Smart Alerts:** Automatically flags items when stock falls below the reorder threshold.
- **Stock Reconciliation:** A dedicated module to manually update stock levels (Add/Remove) with instant UI updates.
- **Search & Filter:** Instant searching functionality to manage large inventories easily.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** Shadcn/UI
- **Icons:** Lucide-React

## 📦 Installation & Setup
1. Clone the repository:
   `git clone https://github.com/your-username/swiftstock.git`
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev`
